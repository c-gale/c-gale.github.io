    ATLAS ADMIN PANEL:

    /// INDEV (FOR 3.0) ///

    Atlas Admin Panel
    Programmer, GUI Designer (Sep 2021 - Current)
    An Admin Panel I've been working on for the past 3 years, In which I've been adding more and more changes over the years as well as revamping the UI.
    The current version (3.0) can currently do the following (plus a lot more that isn't mentioned here):
    • Keep Logs of recently run commands (an output)
    • Custom commands
    • Session Based authentication plus 2 Factor authentication (with SHA-256)
    • GUI equivalent to any command as well as a command bar (cmd bar yet to be added...)
    • Client-Side settings that allow the player to have an easier UI experience
    • A weight system along with customizable roles
    Things that will be added in the future (or being worked on at the moment):
    • (CURRENTLY DOING) 3rd-Party Plugins which could add more functionality to the admin panel
    • (CURRENTLY DOING) A website capable of firing commands to a game server as well as managing other parts of the game without needing to be in-game.
    • (TO DO) Discord Webhooks (through a proxy/website)
    • (TO DO) Automatic Updates from Github
